# NUM-CORE: Unified Numerical Engineering & Simulation Suite
🚀 **Modern Engineering CLI & Visualization Dashboard**

## 📋 Features
- **Root Finder**: Implementations for Newton-Raphson and Simple Iteration.
- **Linear Solver**: A robust system solver with Gauss-Seidel.
- **Data Interpolator**: Newton and Lagrange polynomials.
- **Numerical Calculus**: Advanced integration (Simpson's/Trapezoidal) and Differentiation.
- **Mission Center**: Graphical dashboard for simulation of engineering problems.

## 🛠️ Installation
```bash
git clone https://github.com/omarh-ossam/NUM-CORE.git
cd NUM-CORE
pip install -r requirements.txt
```

## 🖥️ Usage
To start the Command Line Interface:
```bash
python main.py
```
To launch the Graphical Dashboard from the terminal:
`NCC-Terminal> launch`
